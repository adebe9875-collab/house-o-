# houseOS

تطبيق ويب تفاعلي (HTML + CSS + JavaScript) بثلاث أدوات رئيسية:

## 🖥️ مشخص الحاسوب
يسألك عن مواصفات جهازك (الرام، نوع الذاكرة، المعالج، التخزين، الاستخدام) ويقترح لك نظام التشغيل الأنسب.

## 📥 مركز التحميل
قائمة بأنظمة تشغيل متعددة (Windows، Linux بتوزيعاته المختلفة، BSD، Unix، Tails OS) مع روابط تحميل رسمية، بحث وفلترة، ونصائح أمان (SHA256) وحرق الفلاشات.

## ⚖️ مقارنة الأنظمة
جدول مقارنة شامل بين الأنظمة + دردشة صوتية تفاعلية تقارن بين أي نظامين تسألها عنهم.

## 🔒 التحقق من سلامة الأيزو
أداة تحسب بصمة SHA256 لأي ملف ISO محلياً في المتصفح، لمقارنتها بالبصمة الرسمية.

## التقنيات المستخدمة
- HTML5
- CSS3
- JavaScript (Vanilla)

## طريقة التشغيل
افتح ملف `houseOS.html` مباشرة في أي متصفح، بدون أي تثبيت أو سيرفر.

## الترخيص
مفتوح المصدر — استخدمه وعدّل عليه بحرية.
